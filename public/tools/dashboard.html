⚙️ 首次設定（一次做好，約 10 分鐘）
  1. 到 console.cloud.google.com 建一個專案（或用現有的）。
  2. 啟用兩個 API：Google Search Console API 與 Google Analytics Data API（API 和服務 → 程式庫 → 搜尋啟用）。
  3. OAuth 同意畫面：User type 選「外部」，把你自己的 Google email 加進「測試使用者」。
  4. 建立憑證 → OAuth 用戶端 ID → 類型「網頁應用程式」→ 已授權的 JavaScript 來源 加入 https://geoseotoday.com（若本機測試再加 http://localhost）。複製「用戶端 ID」。
  5. 把下面三個欄位填好（存在你瀏覽器本機，不會外流），按「儲存並登入」。

GA4 資源 ID：GA4 → 管理 → 資源設定 → 資源 ID（一串數字）。GSC 網站：網域資源用 sc-domain:geoseotoday.com，網址前綴資源用 https://geoseotoday.com/。
