# 内容负债是什么？90 天 0 点击页面正在拖垮你

**内容负债（content debt）是指网站上「已发布、但长期不产生点击或转化」的页面——最典型的判定标准是 90 天内自然搜索 0 点击。** 这些页面不是中性的存在：它们消耗 crawl budget、稀释站内权重、拉低 Google 对整个域名的质量评价。

## 为什么「放着不管」比你想的更贵

很多站长的直觉是：「这些页面又没害处，放着说不定哪天会有流量。」实务上刚好相反。Google 在官方文档（Search Central 的 crawl budget 说明）明确指出，**低价值 URL 会对网站的爬取与索引产生负面影响**。而在多数内容网站的 GSC 数据里，常见的分布是：**前 10% 的页面贡献超过 80% 的自然点击**，剩下一长条尾巴几乎是零。

当一个站有 500 页、其中 300 页 90 天 0 点击时，Google 看到的是一个「大部分内容没人需要」的网站。这会影响三件事：

1. **爬取排程**：Googlebot 把时间花在爬没人看的旧页，新文章反而排队等索引。
2. **权重分配**：内部链接把 PageRank 平均洒进一堆死页，赚钱页分到的变少。
3. **整站质量信号**：自 Helpful Content 系统并入核心算法后，Google 是以「整站」为单位评估内容质量，一批空壳页会拖累好页面。

## 内容负债的四种常见型态

| 型态 | 特征 | 常见来源 |
|---|---|---|
| 空壳文 | 有标题、有段落结构，但没有任何具体信息 | AI 批量产文未经人工补料 |
| 重复／蚕食页 | 多篇文章打同一个关键词，互相抢排名 | 没有主题群集规划就乱写 |
| 过期内容 | 讲的活动、价格、版本早已失效 | 新闻式、促销式文章 |
| 孤儿页 | 站内没有任何链接指向它 | 改版后遗留、标签页、测试页 |

## 怎么判定一页是不是内容负债？

用 Google Search Console 的成效报表即可，不需要付费工具：

1. 打开 **成效 → 搜索结果**，日期设「过去 3 个月」。
2. 切到「网页」分页，按点击数由小到大排序。
3. 点击 = 0 且曝光极低（例如 < 50）的页面，就是第一批嫌疑犯。
4. 再对照 **索引 → 网页** 报表里的「Crawled - currently not indexed」与「Discovered - currently not indexed」——出现在这两区的页面，代表 Google 已经用行动告诉你：它不觉得值得收录。

## 找到之后怎么办？三选一

每一页 0 点击页面，只有三条路：

- **合并**：主题有价值但内容太薄、或和其他页重复 → 并入主题页或支柱页，原网址 301 跳转过去。
- **移除**：主题本身就没人搜、也没有转化价值 → 直接删除（回 410 或 404），并清掉指向它的内部链接。
- **补真材料**：主题对、意图对，只是内容空 → 补上真实案例、价格区间、规格限制、客户实际会问的问题，让它从空壳变成有血有肉。

## 「先修最碍事的 30 页，再新增内容」

这是处理内容负债最重要的顺序原则。多数人发现流量停滞时的反射动作是「再多写一点」，结果是往一个漏水的桶子里加水。正确做法：

1. 从 GSC 导出 90 天 0 点击页面清单。
2. 按「曝光量 × 主题重要性」排序，挑出**最碍事的 30 页**——通常是曝光有一点但点击挂零、又刚好卡在你重要关键词上的页面。
3. 这 30 页逐一执行合并／移除／补料。
4. 处理完、观察 2–4 周索引与排名变化后，才开始新增内容。

每季重跑一次这个循环，内容负债就不会再堆积。

## 常见问题（FAQ）

**Q1：删掉一堆页面，网站规模变小，排名会不会反而掉？**
不会因为「页数变少」而掉。Google 评估的是质量而非数量；实务上大规模剪枝（content pruning）后，剩余页面的平均排名经常上升，因为爬取资源与内部权重都更集中。前提是删的确实是没流量、没外链、没转化的页。

**Q2：90 天 0 点击是绝对标准吗？**
不是，它是一个实用的起点门槛。季节性内容（例如报税、节庆）要拉长观察窗口到一年；刚发布不满 90 天的新文章也不算负债。

**Q3：页面有曝光但 0 点击，算内容负债吗？**
算「轻度负债」，但处理方式不同——有曝光代表 Google 认得这页的主题，问题多半出在标题与 meta description 不吸引人，先改写这两项再观察，不必急着删。

**Q4：内容负债和 GEO（AI 引用）有关系吗？**
有。AI 引擎挑选引用来源时同样偏好高信任网站；一个充满空壳页的站，被 ChatGPT 或 Perplexity 引用的概率也更低。清负债是 SEO 和 GEO 的共同地基。
