# GSC 三个地方揪出内容负债（附操作）

**要找出网站的内容负债，只需要 Google Search Console 的三个地方：效果报表（筛 90 天 0 点击页）、网页收录报表（看 Crawled / Discovered - currently not indexed 两个状态）、网址检查工具（逐页确认 Google 怎么看它）。** 全程免费，30 分钟可以完成一轮审计。

## 开始前：你需要什么

只需要一个已验证的 GSC 资源（property），且网站上线超过 90 天——因为我们的核心判定标准就是「90 天 0 点击」。2026 年的 GSC 界面效果数据保留 16 个月，足够做季度审计与年度回顾。

## 第一站：效果报表——揪出 90 天 0 点击页

这是最重要的一站，实际操作步骤：

1. 左侧菜单点 **「效果」→「搜索结果」**。
2. 上方日期筛选器改成 **「过去 3 个月」**（90 天）。
3. 下方表格切换到 **「网页」** 标签页。
4. 点「点击」列标题，由**小到大**排序。
5. 右上角 **「导出」→ Google 表格**，把整份清单拉出来。

在表格里加一列「判定」，用两个条件分类：

- **点击 = 0、展示 < 50**：重度负债。Google 几乎不给这页任何机会，走合并或移除。
- **点击 = 0、展示 ≥ 50**：轻度负债。Google 有让它出现在搜索结果，但没人想点——通常是标题与描述的问题，优先改写标题标签与 meta description 再观察。

注意一个陷阱：GSC 的网页标签页默认最多显示 1,000 行。大站要用「网页」筛选器分目录（例如 `/blog/`）分批导出，才不会漏掉长尾。

## 第二站：网页收录报表——Google 已经投票的页面

左侧菜单点 **「收录」→「网页」**（Page indexing 报表），往下滚到「网页未编入索引的原因」清单。重点看两个状态：

| 报表状态 | Google 的意思 | 对你的意义 |
|---|---|---|
| **Crawled - currently not indexed**（已抓取，目前未编入索引） | 我爬了，读完了，决定不收录 | 质量判决：内容不值得进索引，是最明确的负债信号 |
| **Discovered - currently not indexed**（已发现，目前尚未编入索引） | 我知道这页存在，但不急着爬 | 抓取优先度太低——整站质量或内部链接出了问题 |

操作：点进每个状态可以看到 URL 示例清单（同样可导出）。把这两份清单和第一站的 0 点击清单交叉比对——**同时出现在「0 点击」和「Crawled - currently not indexed」的页面，就是最该优先处理的核心负债**。

如果「Discovered - currently not indexed」的数量占比很高（例如超过已收录页的 30%），问题通常不在单页而在整站：crawl budget 被浪费、或内部链接结构让 Google 觉得这些页不重要。这时要回头看整站的抓取预算怎么分配。

## 第三站：网址检查工具——逐页确诊

对前两站筛出的重点页面（建议先处理 30 页），用最上方的**网址检查（URL Inspection）**逐一贴入 URL，确认三件事：

1. **「网址在 Google 服务中」还是「不在」**——确认收录状态与报表一致。
2. **抓取日期**——上次爬取是几个月前？越久没爬，代表 Google 越不在乎这页。
3. **引荐网页（Referring page）**——Google 是从哪里发现这页的？如果来源是奇怪的参数页或标签页，代表你的站内链接结构在制造垃圾路径。

## 三站数据合并后的行动表

把三个来源整理成一张决策表：

- 0 点击＋已收录＋主题重要 → **补真材料**（真实案例、价格区间、规格限制）。
- 0 点击＋Crawled - currently not indexed＋主题和其他页重复 → **合并＋301**。
- 0 点击＋没外链＋主题根本没人搜 → **移除**。

三选一的意义就是不再对每个页面一视同仁：重复的合并、能救的补料、没用的移除。

## 多久跑一次？

**每季一次**。把上述流程做成固定的季度仪式：导出 → 分类 → 处理 30 页 → 记录。连续跑 3–4 季后，多数网站的「Crawled - currently not indexed」数量会明显下降，新文章的收录速度也会变快。

## 常见问题（FAQ）

**Q1：「Crawled - currently not indexed」的页面会自己好吗？**
少数会（Google 偶尔重新评估），但等待不是策略。这个状态本质上是质量判决，不动内容它大概率一直躺在那里。

**Q2：GSC 数据和 GA4 的流量数字对不上，该信哪个？**
找内容负债时信 GSC。GA4 统计所有渠道的会话，GSC 才是「自然搜索点击」的第一手数据；两者定义不同，数字本来就不会一致。

**Q3：新文章发布两周就出现在「Discovered - currently not indexed」，算负债吗？**
不算，这在新站很常见。给它 4–8 周，同时从高权重页面（首页、支柱页）加内部链接拉抬抓取优先度。

**Q4：可以直接把 0 点击页全部 noindex 吗？**
不建议一刀切。noindex 只是把页面藏起来，没有回收它的内部链接权重，也没有解决内容质量问题。三选一（合并／移除／补料）才是根治。
