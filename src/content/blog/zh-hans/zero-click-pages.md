# 0 点击页面该合并、删除还是补料？决策流程

**一个 90 天 0 点击的页面只有三种处理方式：主题有价值但和其他页重复 → 合并；主题没人搜也没转化价值 → 移除；主题对但内容太空 → 补真材料。** 判断顺序是先问「这个主题值不值得存在」，再问「该由哪一页来承载它」。

## 为什么一定要三选一，不能放着？

因为 0 点击页面有持续成本。它占用 Googlebot 的抓取配额（Google 官方文档明确说低价值 URL 影响 crawl budget）、稀释内部链接权重，而且在整站质量评估中扮演扣分项。实务经验上，一个 300 页的博客若有 200 页长期 0 点击，处理掉这 200 页后，剩余页面的索引速度与排名通常在 4–8 周内可观察到改善。

先用 GSC 拉出清单，然后对每一页跑下面的流程。

## 决策流程：四个问题

依序回答四个问题，答案会把你导向三条路之一：

1. **这个主题有人搜、或对转化有帮助吗？**
   - 都没有 → **移除**。
   - 有 → 下一题。
2. **站上有没有另一页在打同一个（或更大的）主题？**
   - 有，而且那页表现更好 → **合并**进那一页。
   - 没有 → 下一题。
3. **这页的内容有具体信息吗？**（数据、步骤、案例、价格——任何 AI 写不出来的东西）
   - 没有，只是「文章形状的空壳」 → **补真材料**。
   - 有 → 下一题。
4. **内容具体但还是 0 点击？**
   - 检查是不是标题／meta 不吸引人（有曝光无点击）、或关键词选错（连曝光都没有）。这属于优化问题，不是负债处理。

## 路线一：合并（consolidate）

适用：多篇薄文打同一主题、互相蚕食，或一篇旧文的内容可以并进支柱页。

操作步骤：

1. 选定「存活页」——通常是曝光较高、外链较多、或层级较高的那一页。
2. 把被并页面中**还有价值的段落**（独特数据、案例、FAQ）搬进存活页，不是全文粘上。
3. 被并页面设 **301 重定向**到存活页。注意：301 要指向「主题最相关」的页，不要全部丢给首页——Google 把大量指向首页的 301 当成 soft 404 处理。
4. 更新站内所有指向旧 URL 的内部链接，直接改链存活页（不要依赖 301 跳转）。
5. 从 sitemap 移除旧 URL。

## 路线二：移除（remove）

适用：主题本身没有搜索需求、没有外链、没有转化角色——例如过期活动页、测试页、自动生成的标签页。

操作步骤：

1. 确认没有外部链接指向它（GSC 的「链接」报表或任何外链工具）。有外链的页面改走合并路线，把链接权重 301 到相关页。
2. 删除页面，让服务器返回 **410 Gone**（比 404 让 Google 更快移出索引；404 也可以，只是慢一点）。
3. 清掉站内指向它的内部链接与 sitemap 条目。
4. 不需要用 GSC 的「移除网址」工具——那是临时遮蔽用的，不是删页流程的一部分。

## 路线三：补真材料（refill）

适用：主题对、意图对、站上也只有这一页在打，但内容是空壳。这是三条路里**唯一会增值**的一条，也最花力气。

「真材料」指的是 AI 凭空写不出来、竞争对手懒得放的东西：

- **真实案例**：你实际做过的项目、踩过的坑、before/after 数字。
- **价格区间**：这件事外包要多少钱、工具月费多少——用户最想知道、大家最不敢写。
- **规格与限制**：什么情况下不适用、上限在哪里。
- **客户会问的问题**：把业务邮箱里真实出现过的问题做成 FAQ。

补料后记得更新 `date_updated`、用网址检查工具请求重新抓取，并从相关文章加 1–2 条内部链接进来。

## 优先顺序：先修最碍事的 30 页

不要试图一口气处理 200 页。按「曝光量 × 主题重要性」排序，先处理 30 页，观察 2–4 周，再处理下一批。**在负债清完之前，暂停新增内容**——往漏水的桶子加水没有意义。

## 常见问题（FAQ）

**Q1：删除或合并后，多久看得到效果？**
索引层面 2–6 周（旧 URL 移出索引、重定向生效）；排名与抓取效率层面通常 1–3 个月。变化幅度取决于负债占比——负债越重，清完后改善越明显。

**Q2：301 重定向要保留多久？可以之后拿掉吗？**
Google 官方建议至少保留一年，让信号完整转移。实务上如果没有技术负担，永久保留最安全，因为外部可能还有旧链接。

**Q3：补料和重写有什么区别？**
补料是在既有 URL 上增加具体信息，保留原本累积的索引历史；重写如果连 URL 都换，等于砍掉重练。除非原文方向完全错误，否则优先在原 URL 上补料。

**Q4：页面才发布一个月就 0 点击，要处理吗？**
不用。90 天是最低观察窗口，新页面给它一季的时间；季节性主题再拉长到一年。
