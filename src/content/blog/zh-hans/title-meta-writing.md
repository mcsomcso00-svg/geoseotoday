# 标题标签与 meta description 怎么写

**title 标签是排名信号，写法核心是「关键词前置＋讲清楚页面内容」，简体建议 30 字内；meta description 不是排名因素，它的任务是提高点击率，用 80–120 字写出这页能解决什么问题。** 两者是搜索结果页上你唯一能控制的广告文案。

## 两者的分工：一个给算法，一个给人

| 项目 | title 标签 | meta description |
|---|---|---|
| 是排名因素吗 | 是（Google 官方确认的轻量信号） | 否（官方多次确认不影响排名） |
| 主要任务 | 告诉 Google 这页在讲什么 | 说服搜索者点你而不是点别人 |
| 显示长度 | 桌面端约 600px（简体约 30 字） | 约 2 行（简体约 80 字内完整显示） |
| 被 Google 改写的概率 | 高（官方说明会依查询重写） | 更高（经常直接抓正文片段） |

## title 标签的写法规则

1. **核心关键词放前面**：越前面的词，权重信号与用户扫视的注意力都越高。「内容负债是什么？90 天 0 点击页面正在拖垮你」优于「你的网站正在被一种叫内容负债的东西拖垮」。
2. **长度控制在 30 个中文字内**：Google 依像素宽度截断（桌面端约 600px），中文字宽，超过就变「…」。截断不会被处罚，但重要信息要在前段讲完。
3. **一页一个主关键词**：一个 title 想塞三个关键词，通常三个都排不好——这是主题集群规划的问题，不是标题技巧问题（见〈主题集群怎么规划〉）。
4. **加差异化元素**：数字（「30 个必查项目」）、年份（2026）、具体承诺（「附操作」「附范例」）。搜索结果上你和九个对手并排，一模一样的标题没有点的理由。
5. **品牌名放尾端、可省略**：「文章标题｜品牌名」是标准格式；长尾文章空间不够时可拿掉品牌名。
6. **每页唯一**：全站重复的 title 是最常见的技术债，用爬虫工具（Screaming Frog 免费版）一次抓出来。

三个可直接套的模板：

- 定义型：`[关键词]是什么？[一句话价值主张]`
- 教学型：`[关键词]怎么做？[数字]个步骤（附[范例/清单/模板]）`
- 清单型：`[数字]个[关键词][错误/工具/技巧]，[痛点或承诺]`

## meta description 的写法规则

1. **80–120 字，前 80 字讲完重点**：移动端显示更短，关键信息前置。
2. **写「这页替你解决什么」，不是「这页包含什么」**：「三步骤用 GSC 找出拖垮排名的 0 点击页面，附实际报表名称与操作路径」优于「本文介绍 GSC 的使用方法」。
3. **包含关键词**：不为排名，而是因为 Google 会把查询词**加粗**显示——粗体字提高被看见的概率。
4. **放具体元素**：数字、方法名、可交付物（清单、模板、流程图）。
5. **每页唯一、与正文相符**：描述说有「价格比较表」正文却没有，换来的是跳出。

## Google 改写你的标题怎么办？

自 2021 年的标题改写更新后，Google 会依查询语境替换显示标题，官方数据约有 **13%** 的情况不用你的 title 原文。常见触发原因与对策：

- **title 太长被截** → 缩短到重点前置。
- **关键词堆砌** → 改写成一句自然的话。
- **title 与 H1、正文不一致** → 让 title 与 H1 保持高度一致（可以不同，但主题要相同）。
- **全站模板化标题** → 每页手写差异化。

改写无法完全避免，但以上四点能把概率压到最低。

## 用 GSC 找出「该改标题」的页面

这是最被低估的免费优化：

1. 进 GSC「效果 → 搜索结果」，日期设 90 天，勾选显示「展示次数」与「点击率」。
2. 切到「网页」标签页，找出**展示高但 CTR 明显偏低**的页面（例如展示 > 1,000、CTR < 1%）。
3. 点进该页看「查询」标签页——用户实际用什么词搜到你，title 有没有回应那个意图？
4. 改写 title 与 description，记录日期，4 周后回来比对 CTR。

这招对「有展示、0 点击」的轻度内容负债页特别有效。

## 常见问题（FAQ）

**Q1：title 和 H1 可以不一样吗？**
可以，Google 也确认两者可分工——title 给搜索结果页（短、带钩子），H1 给进站读者（可更完整）。但主题必须一致，差太多会提高被改写概率。

**Q2：meta description 不影响排名，那可以不写吗？**
可以不写，Google 会自动抓正文——但你就放弃了控制权。重要页面（支柱页、服务页、高展示文章）一定手写；长尾农文可接受由「答案优先」第一段自动充当。

**Q3：关键词要在 title 出现几次？**
一次就够。出现两次以上是堆砌信号，反而触发改写。

**Q4：改了 title 多久生效？**
Google 重新爬取后即更新显示，通常几天到两周。可用 GSC 网址检查工具请求重新检索加速。
