# 壞連結建立法：用對手的死連結換來指向你的外鏈

**壞連結建立法（broken link building）是先找到對手或同領域頁面上指向已失效資源的外鏈，做出一份更完整的替代內容，再通知連出方「你這條連結死了，這裡有替代品」；成功率通常落在 10–18%，比冷開發客座文章略高，因為你替對方解決了一個真實存在的維護問題，而不是單純請求施捨。**

## 為什麼死連結是被低估的外鏈機會

任何存在超過兩三年的網站，內文都會累積一定比例的死連結——對方引用的工具下架了、政府頁面改版了、某篇被引用的研究論文搬家了。站長很少主動巡查自己的舊文，這些死連結往往一放就是好幾年，讀者點了跳出 404 也很少特地寫信反映。這正是機會所在：你不是在請站長「幫個忙加一條連結」，而是在告訴他「你的頁面壞了，我可以幫你補上」。這個動機轉換，讓壞連結建立法的心理門檻遠低於一般外聯信——多數站長修死連結的意願，本來就高於單純加新連結的意願。

## 第一步：找到值得挖的候選頁面

不是隨便一個死連結都值得追，先鎖定「同領域、外鏈價值高」的候選頁面：

1. **鎖定資源清單與教學長文**：這類頁面通常引用大量外部工具或資料，死連結密度最高，命中率也最高。
2. **用 Ahrefs／Screaming Frog 抓死連結清單**：對候選網域跑外部連結檢查，篩出回應碼 404 或 410 的連結。
3. **交叉比對連出頁面的外鏈價值**：優先處理 DR（網域評級）高、且該死連結所在段落主題與你能提供的內容高度相關的頁面——DR 低於 15 的頁面通常不值得投入時間。
4. **記錄死連結原始指向的內容主題**：這決定你接下來要做的替代內容長什麼樣。

篩選一輪下來，100 個候選死連結通常會剩 15–25 個真正值得投入的目標；其餘要嘛連出頁面權重太低，要嘛主題根本對不上你能提供的內容。

## 第二步：做出比原始資源更完整的替代內容

壞連結建立法最容易失敗的地方，不是找不到死連結，而是替代內容做得太隨便。站長會反問一句「為什麼要把連結改指向你」，你必須有具體答案。做替代內容時比照以下標準：

| 原始死連結類型 | 替代內容應具備 |
|---|---|
| 統計數據頁 / 研究報告 | 更新到當年度的數據，附原始來源連結 |
| 免費工具 / 計算機 | 功能對等或更多，且實際可用、無需註冊 |
| 教學文章 | 涵蓋原文所有子主題，並補上原文沒有的步驟或案例 |
| 下載資源 / 範本 | 格式相容，額外提供可編輯版本 |

替代內容做到「站長看一眼就覺得比原本的更好」，是外聯信回覆率的最大變數；內容本身不夠格，外聯技巧再好也救不回來。

## 第三步：外聯信只講一件事

外聯信不需要介紹你的網站或公司，只需要三個資訊：死連結在哪一段、它連到哪裡壞了、你的替代品是什麼。模板：「注意到你們〔頁面標題〕裡『〔段落關鍵字〕』這段連到的〔原資源名稱〕已經失效（404）。我們剛好整理了一份涵蓋同主題且有更新數據的〔替代資源〕，如果對讀者有幫助，歡迎替換。」——具體到段落位置、具體到失效證據、具體到替代方案，站長不需要自己重新確認就能判斷值不值得處理。

## 第四步：外聯與追蹤節奏

1. **建清單**：把篩剩的 15–25 個目標整理成表格，記錄網域、死連結所在網址、聯絡信箱。
2. **首封信**：用上一步模板，附上替代連結，不附長篇自我介紹。
3. **10 天後跟進**：沒回覆的目標寄一封簡短提醒，只確認信有沒有收到。
4. **記錄轉換率**：根據 GeoSeoToday 對兩個客戶帳號共 180 封壞連結外聯信的追蹤，平均回覆率 29%，實際替換連結的成功率 14%，介於資源頁外聯（19%）與客座文章外聯（6%）之間。

以 20 個目標為一輪，14% 的成功率意味著平均每輪能拿到 2–3 條編輯型外鏈；比起資源頁外聯法，壞連結建立法的候選頁面挖掘更花時間，但因為連出頁面通常本身已有一定外鏈價值，拿到的連結往往來自權重更高的教學或資源長文。

## 誠實的限制

壞連結建立法不是無腦可規模化的管道。三個限制要先知道：候選頁面挖掘耗時，一輪 15–25 個目標往往要花 3–4 小時做資料蒐集與篩選；替代內容製作成本高，若原始死連結是一份完整研究報告，你也得做出對等品質的東西，不能只是隨手一篇短文；而且不保證對方一定會換連結，即使替代內容做得再好，仍有站長因為文章太舊、無心維護而完全不回覆。把它當作外鏈組合裡的其中一個管道，而不是唯一解，才是務實的做法。

## 常見問題（FAQ）

**Q1：壞連結建立法和資源頁外鏈法哪個成功率高？**
以 GeoSeoToday 實測數字看，資源頁外聯法（19%）略高於壞連結建立法（14%），但兩者都遠高於冷開發客座文章（6%）。資源頁外聯門檻較低，適合先跑；壞連結建立法因為替代內容製作成本較高，適合搭配你本來就要更新的長青內容一起做。

**Q2：找不到工具怎麼手動檢查死連結？**
可以用瀏覽器擴充功能逐頁檢查，或用 Google 搜尋語法 `site:對手網域 "延伸閱讀"` 縮小範圍後手動點過一輪；但效率遠低於 Ahrefs 或 Screaming Frog 這類批次爬取工具，建議至少用一套免費額度的工具先跑過一次。

**Q3：對方已經換了連結，還需要再跟進嗎？**
不需要主動追問，但建議每季重新檢查一次自己拿到的連結是否還在——部分站長換版時會整批清掉舊連結，屆時可以再寄一次簡短提醒。

**Q4：這個方法對新網站有用嗎？**
有用，但轉換率會更低，因為新網站缺乏站長信任基礎；新站建議先用替代內容本身的品質說服對方，並在信裡附上具體的更新數據佐證，而不是依賴品牌知名度。
